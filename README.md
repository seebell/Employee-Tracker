# Employee-Tracker

## Description
This application is designed for building a solution for managing a company's employees using node, inquirer, and MySQL. This app makes it easy for non-developers to view and interact with information stored in databases.

The command line application allows users to:
* Add departments, roles, employees
* View departments, roles, employees
* Update employee roles
* View employees by manager
* Delete departments, roles, and employees
* View the total utilized budget of a department -- ie the combined salaries of all employees in that department
